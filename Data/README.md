# PoxNetX — `Data/` Directory

This directory documents and provides reproducible access to the dataset artifacts associated with **PoxNetX: A Five-Class Explainable Ensemble Deep Learning Framework for Differential Diagnosis of Viral Pox Skin Lesions**. It does not host raw image files directly; the images themselves are distributed through Mendeley Data (see below) to keep this repository lightweight and to respect the dataset's Creative Commons license terms.

## 1. Dataset Access

The distributable artifact, consisting of the DermGAN-DDPM synthetic augmentation outputs, is published on **Mendeley Data**:

- **DOI:** [10.17632/p4bst76r7w.1](https://doi.org/10.17632/p4bst76r7w.3)
- **Direct link:** https://data.mendeley.com/datasets/p4bst76r7w/3
- **Version:** V3

Only the synthetic outputs generated by the DermGAN-DDPM pipeline are redistributed as a standalone artifact. The underlying real-world source images used to train the generative pipeline are **not** redistributed here; they remain accessible through their original repositories, listed individually in [`primary_sources.md`](./primary_sources.md).

> **Note on scope.** The Mendeley release covers synthetic outputs only. The five-class ensemble classifier (EL03) reported in the associated thesis and journal paper was trained on `Dataset_Raw` (2,611 real images; 2,088 train / 523 validation after an 80/20 stratified split), not on the DermGAN-DDPM augmented corpus. The augmented corpus fed a separate 20-architecture transfer learning benchmark. Please preserve this distinction in any downstream description of the dataset's role.

## 3. Primary Source Datasets

The DermGAN-DDPM pipeline was trained on real lesion images drawn from 16 external sources. Each is listed with its disease class coverage, license, and access link in [`primary_sources.md`](./primary_sources.md). Please consult that file, and the original source terms, before using any upstream imagery directly.

## 6. Data Generation Pipeline

The `generation_pipeline/` folder contains the code implementing the four-stage DermGAN-DDPM hybrid generative augmentation pipeline described in the thesis and journal paper: preprocessing and ROI annotation, a convolutional autoencoder, a latent diffusion model (DDPM), and a Pix2Pix refiner with quality filtering. Pipeline runs are organized per disease class, since each class was trained and generated independently. See [`generation_pipeline/README.md`](./generation_pipeline/README.md) for the stage-by-stage breakdown and file mapping.

Add additional class subfolders (e.g. `monkeypox/`, `chickenpox/`) alongside `cowpox/` and `measles/` as those notebooks become available.
