# Primary Source Datasets

The DermGAN-DDPM generative pipeline was trained on real lesion images drawn from 16 external source datasets. This table lists each source, its disease class coverage, its license, and its access link. It mirrors the "Related Links" section already prepared for the Mendeley Data listing; copy those 16 entries in here directly so the GitHub repository and the Mendeley record stay consistent.

None of the original third-party images are redistributed in this repository. Access each source through its own link and observe its individual license terms.
